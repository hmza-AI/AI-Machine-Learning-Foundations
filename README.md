Machine Learning Foundations & Applications
This repository contains my academic and practical journey in Artificial Intelligence, showcasing various implementations of Machine Learning algorithms applied to real-world datasets.

Core Highlights
1. Academic Admission Predictive Suite
Linear Regression: Predicted the probability of university admission based on GRE and CGPA. (student_admission_linear.ipynb)

Logistic Regression: Binary classification of admission outcomes. (student_admission_logistic.ipynb)

Dataset: Specialized graduate admission data (adm_data.csv).

2. Supervised Learning Portfolio
Support Vector Machines (SVM): Comparative study of different kernels for optimal data separation. (SVM.ipynb)

Decision Trees & Random Forests: Implementation of tree-based models for complex classification. (decision_tree.ipynb)

Multi-Class Classification: Benchmarking KNN and Naive Bayes on benchmark datasets. (classification.ipynb)

3. Unsupervised Learning
K-Means Clustering: 3D visualization of clusters and performance evaluation using Silhouette Scores. (clustering_kmeans.ipynb)

Tech Stack
Core: Python

Libraries: NumPy, Pandas, Scikit-learn

Visualization: Matplotlib, Seaborn
